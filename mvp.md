# MVP Scope

## MVP Goal
Ship a usable MVP for one property that Jennifer can test in real operations.

## Admin / Manager View

### 1. Dispatch
Includes:
- Daily Brief
- Watchlist

Purpose:
- set context for the day
- capture important notes and operational issues

### 2. Task / Card System
Includes:
- create task/card
- assign user
- due date and optional due time
- priority
- draft vs publish

Purpose:
- manage execution

### 3. Templates
Includes a small set of repeatable templates such as:
- Arrival Prep
- Stayover
- Departure Clean
- Deep Clean
- Issue

Purpose:
- reduce manual task creation

### 4. Checklist Enforcement
Includes:
- checklist items on cards
- optional requirement that all checklist items be completed before marking card done

Purpose:
- ensure work is done correctly, not just marked complete

### 5. Activity
Includes:
- simple activity card/feed
- most recent operational changes

Purpose:
- show what changed recently

### 6. Staff
Includes:
- add staff
- remove staff
- assign role
- view staff list
- click into staff card
- notes box
- outcomes feed

Purpose:
- preserve operational memory and accountability

### 7. Card Detail View
Includes:
- title
- status
- assignee
- priority
- description
- checklist
- comments
- attachments
- Mark Complete
- Block / Need Help

Purpose:
- central place where work gets executed

### 8. Card Engagement Tracking
Track and store:
- opened
- commented
- completed
- blocked

Purpose:
- support outcomes and later Admin Cards

### 9. Admin Cards
System-generated management action items, for example:
- overdue tasks
- repeated blockers
- rooms needing deep clean
- unresolved critical issues

Purpose:
- replace reports with action

## Operator / Staff View

### 1. Login
Simple login to access assigned work.

### 2. Daily Brief Card
Visible each day so staff can understand what matters.

### 3. Since Last Shift
Show what changed since the user last worked or logged in.

### 4. Assigned Card Interaction
Staff can:
- open assigned card
- comment
- attach
- complete
- note blocker
- request help

### 5. Create New Card Upward
Staff can submit a simple note and optional attachment upward to management.

## Explicitly Out of Scope
- chat
- deep reporting dashboards
- complex scheduling engine
- recruiting workflows
- payroll/timeclock system
- geo-tracking
- deep ResNexus sync
- AI card generation
