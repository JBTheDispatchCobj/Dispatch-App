# AGENTS.md

## Product

This repository is **Dispatch**: a mobile-first hotel operations MVP for boutique and operator-led properties. It reduces day-to-day chaos by structuring execution, assignments, and visibility.

## MVP direction

Ship the smallest usable version for one property first. Prefer calm, list-oriented UX over noisy dashboards.

## Build rules

- Build the smallest working slice first; avoid scope creep.
- Mobile-first layouts and touch-friendly controls.
- Do not refactor unrelated code when fixing something specific.
- Prefer clear, boring code over clever abstractions.
- End each meaningful change with something you can run and verify locally.

## In scope (high level)

Authentication, dispatch context, tasks/cards, templates, checklists, activity, staff basics, and admin signals—as needed for the MVP, implemented incrementally.

## Out of scope (unless explicitly requested)

Chat, deep PMS integration, AI automation, full HR/payroll, geo tracking, and native app shells beyond responsive web.
